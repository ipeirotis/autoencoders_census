  <files>frontend/server/config/gcp-clients.ts</files>
  <action>
Create new file frontend/server/config/gcp-clients.ts exporting singleton GCP client instances.

Pattern (from Phase 03 research Pattern 5):
```typescript
import { Storage } from '@google-cloud/storage';
import { Firestore } from '@google-cloud/firestore';
import { PubSub } from '@google-cloud/pubsub';
import { env } from './env';

// Create singleton instances
export const storage = new Storage({ projectId: env.GOOGLE_CLOUD_PROJECT });
export const firestore = new Firestore({ projectId: env.GOOGLE_CLOUD_PROJECT });
export const pubsub = new PubSub({ projectId: env.GOOGLE_CLOUD_PROJECT });
```

This prevents connection pool exhaustion from creating multiple client instances across routes (FE-18).

</task>
